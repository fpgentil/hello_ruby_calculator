# HelloRubyCalculator

Simplest gem ever to introduce Ruby in the `Hello Ruby` course.
Slides can be found here: https://speakerdeck.com/fpgentil/hello-ruby

## Installation

```
  $ gem install hello_ruby_calculator
```
